Storage Manager

Institute of Information Systems, Sierre (CH)

The goal of this project was to implement the visualization system of the storage system installed at Technopôle Sierre/Siders. Photovoltaic production counters and consumption counters are integrated and manage the storage Leclanché.

An iterative approach was used to integrate and implement the storage manager system :
Project Planning with Kanban Analysis of different counters and integrate them. Creation of the cumulative curve and implement the algorithms to drive the storage manager. Implementation of a modbus TPC/IP interface to send command to the storage manager. Visualization coached by Prof. Yann Bocchi

Florify

Gamified gardening App

Florify is Farmville for real life - a mobile app that makes gardening easy, engaging, and fun to help urban casual gardeners.

Florify's engaging user experience includes: 1) Step by step guides for selecting, planting and growing plants; recommendations are customized for user's preferences, gardening experience / expertise, temperature, sunlight and humidity. 2) Reminders to water and care for the plants through push notification 3) Progress is tracked through images and interest is enhanced through gamification 4) Ordering everything you need to start and maintain your own garden.

  • Date: June 2015
  • Client: Florify
  • Category: Consulting / Project management
